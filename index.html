This page links to a few projects related to understanding the Rigveda.

Project Description Status
Rigveda Sukta Connections Visualizer This project visualizes the relationship between various various suktas across the Rigveda. For example, a sukta in the 1st mandala could be relate to one in the 9th For this relationship, we consider the similarity of text between the mantras which constitute the sukta. WIP
Rigveda Mantra Connections Visualizer This project visualizes the relationship between various various mantras - a finer grained version of the Sukta Visualization project. WIP
Devata-Sukta mapper This project allows you to see which devatas are refered to in which suktas. WIP
Chandas-Sukta mapper Similar to the devatas project, this project maps chandas to the suktas. WIP
